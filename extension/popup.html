插件配置

在这里填写 API Key、城市 ID 和搜索关键词，脚本会按配置轮换执行。
每行一个，或使用逗号分隔。获取方式：先在 Boss 职位列表页切到目标城市，地址栏里的 city=101250100 就是 cityId；也可以直接把整段职位列表 URL 粘贴进来，插件会自动提取。
每行一个，或使用逗号分隔。至少保留 1 个关键词，脚本才允许启动。
用于限制从“开始执行”到“停止执行”这一整次运行里，累计最多点击多少个岗位。所有城市和关键词组合共享这个总上限，达到后会自动停止，降低连续点击过多带来的风控风险。
输入岗位方向、技术栈、经验侧重点或业务偏好后，AI 会生成一组更适合 Boss 搜索框的关键词，并合并到上面的关键词输入框中。生成后记得点“保存配置”。
至少填写 1 个城市 ID 和 1 个搜索关键词后，才能开启自动执行；切换组合时插件会先改 URL，待页面加载完成后再强制点击一次搜索。
